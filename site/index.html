Frequently asked

Does LAC require Ollama?

Yes. LAC manages models through Ollama. Install Ollama first — LAC handles the rest.

Is it actually free?

The core — hardware scan, recommendations, chat, web UI, TUI — is MIT open source. Benchmarking and tuning automation are LAC Pro, a paid add-on.

Will my data leave my machine?

LAC has no telemetry. Model requests go to your configured providers; Ollama defaults to this machine, but providers and endpoints may be remote. LAC may check GitHub Releases for updates and automatically refresh stale public model metadata from Ollama.com. Explore web search, configured MCP tools, model imports, and Pro activation create additional documented boundaries. See the Privacy policy for the exact data involved.

What GPUs are supported?

NVIDIA via nvidia-smi, AMD via Vulkan (RX 6800 XT verified), Apple Silicon M-series, Intel Arc. CPU-only works too — just slower.

What does LAC stand for?

Local AI Companion. One command (lac), one job: find the best model your machine can actually run — and tell you how sure it is. (Real-world benchmark/tuning automation is LAC Pro.)

Is there a GUI?

Desktop app (Windows installer) + web UI on :5050 + full terminal TUI. Pick your interface.
